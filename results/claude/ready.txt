## ГОТОВИЙ

---

### 1. Core Language — Типи та змінні

1. Яка різниця між `var x int` та `x := 0` і коли що обирати?
2. Що таке zero value і для яких типів воно `nil`?
3. Яка різниця між `rune` і `byte` при ітерації рядка?
4. Коли `type MyInt int` відрізняється від `type MyInt = int`?
5. Як `iota` використовується для бітових масок?
6. Чому string в Go immutable і що це означає на практиці?
7. Як конвертувати `[]byte` ↔ `string` без зайвих алокацій?
8. Як shadowing змінних у вкладених блоках призводить до багів?

### 2. Core Language — Слайси та мапи

9. Яка різниця між `len` і `cap` у слайсі?
10. Чим `nil` slice відрізняється від порожнього (`[]T{}`)?
11. Що відбувається з оригінальним слайсом після `append` за межі `cap`?
12. Що відбувається при `append` до shared underlying array двох слайсів?
13. Як `a[low:high:max]` контролює `cap` нового слайса?
14. Як безпечно видалити елемент зі слайса без збереження порядку?
15. Чому `map` не є потокобезпечною і що станеться при write в `nil` map?
16. Який порядок ітерації `map` і чи можна на нього покладатися?
17. Як невизначений порядок map впливає на тести та серіалізацію?
18. Чи може struct бути ключем у `map` і які обмеження?

### 3. Core Language — Структури та методи

19. Яка різниця між value receiver і pointer receiver у методах?
20. Коли Go автоматично бере адресу для виклику pointer receiver?
21. Що таке promoted fields і методи в embedded struct?
22. Як вирішити конфлікти методів при embedding?

24. Як порівнювати дві struct — коли компілюється, коли ні?
25. Що станеться, якщо викликати метод на nil receiver?

### 4. Core Language — Інтерфейси

26. Чому interface може бути `not nil` із `nil` pointer всередині?
27. Як безпечно перевірити interface на nil?
28. Що таке type assertion і коли він панікує?
